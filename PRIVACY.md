# Privacy Policy

Discogs Collection Manager is a local-first application for managing a personal Discogs collection.

## Data Stored Locally

The app stores collection data locally on your machine, including release metadata, artists, labels, styles, prices, videos, tracklists, invoice import metadata, and sync status.

This data is stored in the local application database and is not uploaded to this project's developer or to any separate backend service operated by this app.

## Discogs API Usage

The app connects to the Discogs API using the Discogs credentials you configure locally.

Requests to Discogs may include your Discogs username, release IDs, collection operations, invoice-matched release IDs, and other data required to sync or update your collection.

Discogs handles that data under Discogs' own privacy policy and API terms.

## Invoice Imports

If you use invoice import, invoice PDF contents are processed locally by the app to extract line items, invoice number, invoice date, shipping price, and release candidates.

Extracted invoice metadata may be stored locally with imported releases. Invoice files are not intentionally uploaded to any app-operated server.

## No App-Operated Tracking

The app does not include analytics tracking, advertising trackers, third-party tracking cookies, or telemetry operated by this project.

## Cookies And Local Storage

The app does not currently use authentication cookies. Some UI preferences or cached data may be stored in browser local storage.

## Your Responsibility

Because this is a self-hosted/local app, you are responsible for protecting your local database, `.env.local` credentials, and any invoice files you process.

Do not publish your `.env.local`, database files, invoices, or other personal data.
